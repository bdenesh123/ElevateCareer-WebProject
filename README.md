# ElevateCareer

ElevateCareer is a modern employment search website designed to help users find jobs and career opportunities easily.

## Features

- Browse and search for the latest job postings
- View job details including location, salary, and description
- User-friendly interface for easy navigation
- Fully responsive design for desktop and mobile

## Tech Stack

- **Frontend:** React.js, Tailwind CSS
- **Animations:** GSAP
- **Icons:** FontAwesome
- **Deployment:** Vercel
